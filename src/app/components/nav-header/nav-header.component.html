Enjoy!

Tutorial

Step 1

At top right corner, key in number of panel and press generate. only allow 1 - 9.

Step 2

Using keyboard key 1 to 9 to focus the panel. Example: pressing "1" in keyboard will focus to "Panel 1"

Step 3

Type your word to be search from.

Step 3

Press keyboard "Enter" to search the word.
